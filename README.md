# Desafio Técnico Angular - Fabiane Rodrigues Sarres

Projeto desenvolvido para o desafio técnico, contendo um **CRUD de Usuários** com Angular 17+, utilizando boas práticas, componentes standalone, Router e persistência via `localStorage`.

---

Funcionalidades:

- CRUD completo de Usuários
- Formulário reativo (Reactive Forms)
- Validações de campos
- Endereço com UF (select), logradouro e número
- Navegação: /users, /users/new, /users/:id/edit
- Persistência no localStorage
- Estilização com Bootstrap 5 + tema customizado
- Uso das novas Control Flow (@if, @for)
